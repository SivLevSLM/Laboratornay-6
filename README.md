# Лабораторная работа №6 по ОСИР / Игра "Bulls-and-Cows"
Компьютер загадывает случайное число в заданном  диапазоне, а игрок должен угадать его. После каждой попытки  программа сообщает, больше или меньше загаданного числа  предложенное игроком значение. Особенности реализации: Функции для генерации случайного числа,  считывания ввода пользователя и проверки ответа.

## Использование
Установите пакет файлов и распакуйте. 
Откройте терминал git bash (Windows) / Terminal (Linux) и перейди в папку проекта:
```sh
cd guess-number
```
Выдайте права: 
```sh
chmod +x run.sh
chmod +x src/*.sh
```
Для запуска игры: 
```sh
./run.sh
```
